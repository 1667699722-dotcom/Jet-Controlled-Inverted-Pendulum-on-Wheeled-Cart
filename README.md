#Jet-Controlled Inverted Pendulum on Wheeled Cart

A Mathematica-based simulation project for stabilizing an inverted pendulum mounted on a wheeled cart (on frictionless ground) using simplified jet impulse control (jet nozzles installed on both sides of the cart).

#Project Overview

This project takes the classic inverted pendulum-cart system (cart moving on a frictionless surface) as the research object, and explores the feasibility of stabilizing the inverted pendulum (preventing it from falling) by applying simplified jet impulses through nozzles on both sides of the cart.

#Core Features

1.First-Principles Derivation: Directly derive the system's dynamic equations from the Lagrangian (Lagrange function) instead of empirical methods, ensuring theoretical rigor.

2.Symbolic Computation with Mathematica: Leverage Mathematica's powerful symbolic calculation capabilities to:
Precisely derive the nonlinear dynamic equations of the pendulum-cart system
Avoid manual calculation errors in complex mechanical derivation

3.Jet Impulse Control Simulation: Implement simplified jet impulse control logic and conduct numerical simulations to verify the feasibility of using jet forces to stabilize the inverted pendulum.

4.Reproducible Results: All symbolic derivation and numerical simulation code is fully open-sourced, with clear comments for easy reproduction and modification.
